## Project Overview
This project was completed as part of the Deloitte Australia Data Analytics Virtual Experience Program. The objective is to analyze operational telemetry data and workforce equality data to identify machine downtime patterns and potential pay equity issues.

## Business Objectives
- Identify factories with highest machine downtime
- Determine device types contributing most to downtime
- Build interactive Tableau dashboards
- Classify equality scores using Excel

## Tools Used
- Tableau Public
- Microsoft Excel
- JSON
